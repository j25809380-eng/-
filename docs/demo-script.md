# FitNote 项目演示脚本

## 开场

“本项目是一个基于微信小程序的健身记录系统，名称为 FitNote。它在保留高保真设计稿风格的基础上，完成了用户系统、训练记录、训练计划、数据分析、社区互动以及后台管理系统的完整开发。”

## 小程序演示顺序

1. 登录页进入系统。
2. 首页展示今日状态和快捷入口。
3. 进入训练计划页，查看系统计划和我的自定义计划。
4. 点击创建自定义计划，现场新增训练日和动作。
5. 进入训练记录与历史页，展示训练趋势和月度报告。
6. 进入社区页，演示点赞与评论。
7. 进入个人中心，演示资料编辑与头像上传。

## 后台演示顺序

1. 打开后台管理端并登录管理员账号。
2. 展示总览页的核心业务数据。
3. 展示用户管理、训练数据和社区审核。
4. 打开数据大屏，说明系统具备更高层级的数据可视化展示能力。

## 结束总结

“FitNote 不只是一个界面原型，而是一个具备完整业务流程、后端服务、数据库设计、权限控制和后台展示能力的毕业设计项目，能够较好体现软件工程、前后端开发和系统设计能力。”
