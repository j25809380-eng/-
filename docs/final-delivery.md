## 微信登录模式

### 1. 默认模式

默认采用 `mock-login`，适合开发和答辩演示。

配置位置：

```yaml
app:
  wechat:
    mock-login: true
```

### 2. 真实微信登录模式

如果你已经有自己的微信小程序 `appId` 和 `secret`，可改为：

```yaml
app:
  wechat:
    mock-login: false
    app-id: 你的小程序AppId
    secret: 你的小程序Secret
```

后端会调用微信 `code2Session` 完成 `openid` 解析。
